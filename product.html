產品實績

竣碩事業群的產品類別，包括矽膠（Silicone）、橡膠（Rubber）和塑膠及複合材料（Plastic、Composites），我們持續為各個合作夥伴提供這些產品零件，一直以來無論在開發能力、製作技術和品質管制部份，都獲得客戶們的肯定。矽膠類產品包括按鍵、環保產品、婦幼用品、腳墊、襯墊、墊圈、醫療產品等，橡膠類產品包括避震墊、避震器、油封、O形環、工業用零件、車用零件、握把、耐候產品等，塑膠及複合材料產品包括插孔蓋（Jack Cover）、組合按鍵、車用零件、其他開發中複合材料等，都在多年來經驗與技術的累積下，為客戶提供最完善的服務。

  • 矽膠 x 按鍵

    矽膠類按鍵產品通常具有背光性，是無毒且相當環保的製品，具有良好觸感，可與塑膠或金屬等其他材質相結合，且使用效期長。竣碩事業群在包括手機、遊戲機控制器、遙控器等設備領域，都有五年以上良好的矽膠按鍵製作經驗與實績。

  • 矽膠 x 環保產品

    矽膠的主要成份是二氧化矽，是一種由矽土中的矽酸鈉與硫磺酸製成的無定形的機器製成品。矽膠在製作過程有毒成份即已被完全釋放抽離，且本身可耐200度的高溫，可製作杯環、環保矽膠保鮮膜等產品。

  • 矽膠 x 婦幼用品

    矽膠具備無毒環保的特性，許多廠商利用矽膠製作實感矽膠奶嘴、嬰幼兒餵食碗、嬰幼兒學習筷、矽膠圍兜以及嬰幼兒折疊碗等產品，都具有輕巧方便、易收納的特性。

  • 矽膠 x 腳墊

    矽膠在調合配料後具有良好的抗張強度及抗撕裂性，因此具有緩衝防震功能，能有效阻隔外來或自發性的振動，讓機械內部的運轉更加順暢，保護裝置免受長時間累積振動而損耗。

  • 矽膠 x 襯墊

    矽膠具有止滑和吸收震動的特性，矽膠製成的襯墊能夠維持它原有的彈性和形狀，即使經過長久的壓力測試，也不會降低矽膠的吸震特性，適合做為緩衝材質。

  • 矽膠 x 墊圈

    矽膠是介於有機與無機間的聚合物，它的熱穩定性相當良好，在攝氏零下100度到250度之間仍可保持穩定性質，同時具備抗粘、低化學活性以及透氣性，因此它被廣做為機械方面的墊圈。

  • 矽膠 x 醫療產品

    矽膠符合FDA標準要求，在製作加工過程，亦是最佳材料，且矽膠材質具有耐熱性、難燃性、化學的安定性、耐候性、電氣絶縁性以及耐放射線性，可做為醫療零配件或穿戴式醫療產品。

  • 矽膠 x 插孔蓋

    橡膠的插孔蓋具有耐寒、耐高溫、抗臭氧、耐油性、抗老化等特性，同時擁有電氣絕緣性，包括電子設備、一般民生用品都可被應用。

  • 橡膠 x 避震墊

    橡膠為一種能伸長至原來兩倍長度以上且能迅速恢復到原來大小的高分子量的聚合物，一般大略可分為天然橡膠與合成橡膠。橡膠具有優秀的彈性，抗磨擦與抗張力的特性，可被做為避震墊材質使用。

  • 橡膠 x 避震器

    橡膠能有效承載各級荷重及變形量，並隔絕振動噪音，防震效率佳，適用於各類機械設備振動緩衝的設計上，包括電子設備、控制盤、空氣壓縮機、各類機械設備等的避震器零件都可被應用。

  • 橡膠 x 油封

    油封(Oil Seals)顧名思義，油封的作用主要是保持系統中之壓力，防止液壓油及灰塵之洩漏或進入，且由具有抗壓、抗磨、耐熱等特性之人造橡膠或合金材料製成。常見用於工業設備、汽機車零件，以及各種傳動設備上。

  • 橡膠 x O形環

    O形環，是一種圓環形狀的機械墊片，它是環狀的彈性體，斷面常見為圓形，一般會固定在一凹槽中，組裝過程中會被兩個或兩個以上的組件壓縮，因此產生密封的接口。因為價格便宜，製造簡單，功能可靠，並且安裝要求簡單，O形環是最常見的密封用機械設計。

  • 橡膠 x 工業用零件

    許多合成橡膠材質透氣性小，耐熱老化性、耐候性及耐臭氧性能也良好，並同時擁有電氣絕緣性，或具備機械耐磨性、抗張性高，在高硬度性況下仍保有彈性，常做為工業零件及電氣、電子零件使 用。

  • 橡膠 x 車用零件

    橡膠被運用在車用零件上的歷史已經相當久遠，合成橡膠具備耐油、耐溫、耐磨、耐酸鹼、彈力高、無毒、避震、抗靜電及導電之特性，因此非常適合高速運轉的車輛使用。

  • 握把

    許多開發商都會將橡膠特性應用到握把上，其產品物理性表現，無論在其抗撕裂性, 耐候性、耐磨性均無廢料處理的優點。材質表面具備柔軟防滑的親膚性，讓使用者在捉握之間都能獲得舒適的體驗。

  • 橡膠 x 耐候產品

    材料如塗料、建築用塑料、橡膠製品等，應用於室外經受氣候的考驗，如光照、冷熱、風雨、細菌等造成的綜合破壞，其耐受能力叫耐候性。使用橡膠製成的耐候產品，能夠延長產品在戶外的長時間使用，竣碩事業群有良好的應用經驗。
